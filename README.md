# RL Firewall Simulation (Flask + React)

A complete reinforcement-learning-based firewall simulation system.

## Features
- DQN model for packet decision-making  
- Flask backend with WebSocket (Socket.IO)  
- React frontend visualization  
- Real-time firewall decisions (ALLOW / RATE-LIMIT / BLOCK)
